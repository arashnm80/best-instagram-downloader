# Best Instagram Downloader

## How to run guide
- clone repo
- set these to env
```
BEST_INSTAGRAM_DOWNLOADER_BOT_API
LOG_CHANNEL_ID
INSTAGRAM_USER_1
INSTAGRAM_PASS_1
```
- install required python modules:
```
pip3 install -r requirements.txt
```
- run the main file with:
```
nohup python3 best_instagram_downloader.py &
```
